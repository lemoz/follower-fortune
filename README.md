# NetWorkNetWorth (NWNW)

NetWorkNetWorth publishes clearly labeled, speculative estimates of the combined net worth of an X/Twitter account's followers. Curated boards use committed public-source research; unknown handles can request an admission-capped, paid research build. No placeholder or invented board is shown.

## Run locally

```sh
cp .env.example .env
npm install
node server.mjs
```

The app is intentionally zero-framework: `server.mjs` is the Node HTTP backend and `index.html` is the vanilla-JavaScript frontend. `@resvg/resvg-js` renders Open Graph cards.

To exercise every public route without allowing paid research:

```sh
npm test
```

The smoke test sets `NWNW_DISABLE_RESEARCH=1` and uses non-secret sentinel provider keys. It verifies metadata, canonical redirects, robots/sitemap output, public-data sanitization, build-request admission, and sensitive-file denial.

## Launch configuration

- `PUBLIC_ORIGIN` controls canonical, sitemap, and social URLs.
- `GA_MEASUREMENT_ID` enables the consent-gated Google Analytics loader. No Google request is made before the visitor accepts analytics.
- `GOOGLE_SITE_VERIFICATION` injects an optional Search Console verification meta tag.
- `NWNW_DISABLE_RESEARCH=1` disables new on-demand builds without taking researched boards offline.

`robots.txt` and `sitemap.xml` are generated by the server. The sitemap contains only the homepage, privacy page, and completed static/dynamic boards. Public research JSON is sanitized to exclude free-form location and verdict prose and carries an `X-Robots-Tag: noindex` response header.

## Deploy

Pushes to `main` deploy automatically through GitHub Actions to Fly.io. A push is a production release: test locally, obtain release approval, push, and watch the matching workflow run with `gh run watch <id> --repo lemoz/networknetworth`.

All figures are speculative estimates for entertainment and research, not verified facts or financial advice.
